Features:

  • Deposit & Withdrawal transactions
  • Complaint acknowledgement
  • Complaint resolution
  • Cheque clearing
  • Cheque return
  • Account Balance (specify a time or an amount to trigger alerts)

You can request the following information by sending an SMS to 2342

  • Account balance enquiry
  • Internal funds transfer (within your own accounts)
  • Transfer (Third-party funds with DIBPL)
  • Mini statement (indicating last 5 transactions in your account)
  • Cheque book request
  • Bank statement request
  • 3 types of Utility Bills
  • Help menu

Notes:

Balance Alert is not sent on every transaction alert.



Service Description Type Command
PIN Issuance PIN "Numeric PIN"
PIN Change PIN "Old PIN" "New PIN"
Account Balance AB "PIN" "Alias"
Mini Statement MS "PIN" "Alias"
